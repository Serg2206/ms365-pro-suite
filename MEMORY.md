## 3. Шаблоны промптов

### Научные (НАУ-1..НАУ-8)

**НАУ-1: Анализ научной статьи**
```
Проанализируй статью [АВТОР, ГОД] на тему [ТЕМА].
Структура ответа:
1. Краткое содержание (3-5 предложений)
2. Методология
3. Ключевые выводы
4. Критика: сильные и слабые стороны
5. Связь с моей работой по [ТВОЯ ТЕМА]
6. Цитируемые источники (какие работы цитирует)
7. Кто цитирует эту работу (если известно)
```

**НАУ-2: Литературный обзор**
```
Напиши литературный обзор по теме [ТЕМА].
Параметры:
- Объём: [N] слов
- Период: [ГОДЫ]
- Фокус: методологии и выводы
- Язык: [ЯЗЫК]
- Контекст: раздел статьи для [ЖУРНАЛ]

Структура:
1. Введение
2. Основные направления (3-4)
3. Сравнительный анализ
4. Пробелы в литературе
5. Перспективы
6. Выводы
```

**НАУ-3: Формулировка гипотез**
```
Сформулируй гипотезы для исследования [ТЕМА].
Контекст: [ОПИСАНИЕ ИССЛЕДОВАНИЯ]
Переменные: [СПИСОК ПЕРЕМЕННЫХ]

Требования:
- H1: основная гипотеза
- H2-H4: дополнительные гипотезы
- Каждая гипотеза: направленная, проверяемая
- Связь с литературой
```

**НАУ-4: Статистический анализ**
```
Опиши статистический анализ для данных:
[ОПИСАНИЕ ДАННЫХ]

Переменные:
- Независимые: [СПИСОК]
- Зависимые: [СПИСОК]

Требования:
1. Описательная статистика
2. Проверка нормальности
3. Выбор критериев (обоснование)
4. План интерпретации
5. Визуализация (какие графики)
```

**НАУ-5: Тезисы для конференции**
```
Подготовь тезисы для конференции [НАЗВАНИЕ].
Тема: [ТЕМА]
Формат: [устный / постер]
Объём: [N] слов
Требования: [ТРЕБОВАНИЯ КОНФЕРЕНЦИИ]

Структура:
- Актуальность
- Цель
- Методы
- Результаты
- Выводы
- Ключевые слова (5-7)
```

**НАУ-6: Перевод с терминологией**
```
Переведи текст с [ЯЗЫКА] на [ЯЗЫК], сохраняя терминологию:

[ТЕКСТ]

Требования:
- Термины в [область] оставь на языке оригинала
- Приведи глоссарий терминов
- Сохрани академический стиль
```

**НАУ-7: Проверка на плагиат**
```
Проанализируй текст на предмет заимствований:

[ТЕКСТ]

Требования:
1. Выдели потенциально проблемные фрагменты
2. Оцени оригинальность (%)
3. Предложи перефразировку для проблемных мест
4. Проверь правильность цитирования
```

**НАУ-8: Библиография по ГОСТ**
```
Оформи список литературы по ГОСТ 7.1-2003:

[СПИСОК ИСТОЧНИКОВ]

Требования:
- Порядок: алфавитный
- Журналы: авторы. название // журнал. год. том. №. стр.
- Книги: авторы. название. город: издательство, год.
- Проверь: все источники процитированы в тексте
```

### Публицистические (ПУБ-1..ПУБ-8)

**ПУБ-1: Аналитическая статья**
```
Напиши аналитическую статью на тему [ТЕМА].
Издание: [НАЗВАНИЕ]
Объём: [N] знаков

Структура:

- Аргументация (факты > мнения)
- Цитаты экспертов
- Заключение (вывод + перспектива)

Контекст: [ФОНОВАЯ ИНФОРМАЦИЯ]
Источники: [КЛЮЧЕВЫЕ ИСТОЧНИКИ]
```

**ПУБ-2: Убедительный лид**
```
Напиши 5 вариантов вступления (лида) для статьи на тему [ТЕМА].

Требования:
- Каждый вариант: 2-3 предложения
- Разные подходы: вопрос, цитата, факт, история, тезис
- Целевая аудитория: [ОПИСАНИЕ]
- Тон: [нейтральный / критический / оптимистичный]
```

**ПУБ-3: Вопросы для интервью**
```
Подготовь [N] вопросов для интервью с [ЭКСПЕРТ].

- Должность/статус: [ОПИСАНИЕ]
- Тема интервью: [ТЕМА]
- Формат: [по email / телефон / видео]

Структура:
1. Вводные (3-4 вопроса)
2. Основные (5-7 вопросов)
3. Уточняющие (3-4 вопроса)
4. Завершающие (2-3 вопроса)
```

**ПУБ-4: Рерайт новости**
```
Перепиши новость с другого ракурса:

Оригинал: [ТЕКСТ НОВОСТИ]

Требования:
- Сохрани факты
- Измени акцент
- Добавь контекст

```

**ПУБ-5: Колонка/мнение**
```
Напиши колонку на тему [ТЕМА].

Требования:
- Объём: [N] знаков
- Тон: личный, убедительный
- Структура: тезис → аргументация → призыв
- Отличная от новостей: мнение автора главное
```

**ПУБ-6: Адаптация под платформы**
```
Адаптируй текст под разные платформы:

Исходный текст: [ТЕКСТ]

Платформы:
- Telegram (кратко, эмодзи)
- LinkedIn (профессионально)
- Twitter/X (280 символов)
- СМИ (полный формат)
```

**ПУБ-7: Редактура текста**
```
Отредактируй текст:

[ТЕКСТ]

Требования:
- Убери канцеляризмы
- Проверь факты
- Усиль финал
- Добавь динамики
- Проверь логику изложения
```

**ПУБ-8: Заголовки и подзаголовки**
```
Создай для текста:

Требования:
- 10 вариантов заголовков
- 7 вариантов подзаголовков
- 3 варианта лида
- Разные стили: информативный, интригующий, вопросительный
```

## 10. Устранение неполадок

### PowerShell скрипт не запускается
**Решение:** `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

### OneDrive не синхронизируется
**Решение:** Перезапустить OneDrive, проверить свободное место

### VBA макрос не работает
**Решение:** Включить макросы: File → Options → Trust Center → Macro Settings

### Kimi не отвечает
**Решение:** Проверить интернет, обновить страницу, попробовать другой браузер

### Notion не синхронизируется
**Решение:** Проверить API токен, обновить интеграцию

### Git push не работает
**Решение:** Создать Personal Access Token (classic) с правами repo
